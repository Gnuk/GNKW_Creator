[action] [params]
    create   class|test [namespace] [classname]   Create a class

    Example    create class Foo Bar               Create the Foo\Bar class in your source directory
               create test Foo Bar                Create the Foo\BarTest class in your source directory

    Note       Use / instead of \ for [namespace] parameter
